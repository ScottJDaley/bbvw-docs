# ![icon](../img/items/leaf-stone.png) Leaf Stone

**Category:** Evolutionary

## Description
Evolves an Exeggcute into Exeggutor, a Gloom into Vileplume, a Nuzleaf into Shiftry, a Pansage into Simisage, or a Weepinbell into Victreebel.
