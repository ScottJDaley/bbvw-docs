# Bind

**Type:** ![normal](../img/types/normal.png)  
**Category:** ![physical](../img/types/physical.png)  
**Power:** 15  
**Accuracy:** 85  
**PP:** 20  

## Description
Prevents the target from fleeing and inflicts damage for 2-5 turns.
