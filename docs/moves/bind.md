# Bind

**Type:** <img src="img/types/normal.png" width="60" />  
**Category:** <img src="img/types/physical.png" width="50" />  
**Power:** 15  
**Accuracy:** 85  
**PP:** 20  

## Description
Prevents the target from fleeing and inflicts damage for 2-5 turns.
