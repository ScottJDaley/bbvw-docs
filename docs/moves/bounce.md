# Bounce

**Type:** ![flying](../img/types/flying.png)  
**Category:** ![physical](../img/types/physical.png)  
**Power:** 85  
**Accuracy:** 85  
**PP:** 5  

## Description
User bounces high into the air, dodging all attacks, and hits next turn.
