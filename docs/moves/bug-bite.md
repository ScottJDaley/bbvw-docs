# Bug bite

**TM/HM:** 

**Type:** <img src="img/types/bug.png" width="60" />  
**Category:** <img src="img/types/physical.png" width="50" style="object-fit:contain;" />  
**Power:** 60  
**Accuracy:** 100  
**PP:** 20  

## Description
If target has a berry, inflicts double damage and uses the berry.
