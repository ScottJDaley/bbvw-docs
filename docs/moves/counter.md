# Counter

**Type:** ![fighting](../img/types/fighting.png)  
**Category:** ![physical](../img/types/physical.png)  
**Power:** None  
**Accuracy:** 100  
**PP:** 20  

## Description
Inflicts twice the damage the user received from the last physical hit it took.
