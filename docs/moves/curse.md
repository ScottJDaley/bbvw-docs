# Curse

**Type:** ![ghost](../img/types/ghost.png)  
**Category:** ![status](../img/types/status.png)  
**Power:** None  
**Accuracy:** None  
**PP:** 10  

## Description
Ghosts pay half their max HP to hurt the target every turn.  Others decrease Speed but raise Attack and Defense.
