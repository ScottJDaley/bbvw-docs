# Dig

**TM/HM:** TM28

**Type:** ![ground](../img/types/ground.png){ width=60 }  
**Category:** ![physical](../img/types/physical.png){ width=50 style='object-fit:contain;' }  
**Power:** 80  
**Accuracy:** 100  
**PP:** 10  

## Description
User digs underground, dodging all attacks, and hits next turn.
