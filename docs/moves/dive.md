# Dive

**TM/HM:** HM06

**Type:** ![water](../img/types/water.png){ width=60 }  
**Category:** ![physical](../img/types/physical.png){ width=50 style='object-fit:contain;' }  
**Power:** 80  
**Accuracy:** 100  
**PP:** 10  

## Description
User dives underwater, dodging all attacks, and hits next turn.
