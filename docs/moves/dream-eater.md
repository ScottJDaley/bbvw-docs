# Dream eater

**Type:** ![psychic](../img/types/psychic.png)  
**Category:** ![special](../img/types/special.png)  
**Power:** 100  
**Accuracy:** 100  
**PP:** 15  

## Description
Only works on sleeping Pokémon.  Drains half the damage inflicted to heal the user.
