# Echoed voice

**TM/HM:** TM49

**Type:** ![normal](../img/types/normal.png)  
**Category:** ![special](../img/types/special.png){ style='object-fit:contain;' }  
**Power:** 40  
**Accuracy:** 100  
**PP:** 15  

## Description
Power increases by 100% for each consecutive use by any friendly Pokémon, to a maximum of 200.
