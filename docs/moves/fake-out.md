# Fake out

**Type:** <img src="img/types/normal.png" width="60" />  
**Category:** <img src="img/types/physical.png" width="50" />  
**Power:** 40  
**Accuracy:** 100  
**PP:** 10  

## Description
Can only be used as the first move after the user enters battle.  Causes the target to flinch.
