# Fake out

**TM/HM:** 

**Type:** ![normal](../img/types/normal.png)  
**Category:** ![physical](../img/types/physical.png){ style='object-fit:contain;' }  
**Power:** 40  
**Accuracy:** 100  
**PP:** 10  

## Description
Can only be used as the first move after the user enters battle.  Causes the target to flinch.
