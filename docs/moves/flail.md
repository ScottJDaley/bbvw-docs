# Flail

**Type:** ![normal](../img/types/normal.png)  
**Category:** ![physical](../img/types/physical.png)  
**Power:** None  
**Accuracy:** 100  
**PP:** 15  

## Description
Inflicts more damage when the user has less HP remaining, with a maximum of 200 power.
