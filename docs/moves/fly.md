# Fly

**Type:** ![flying](../img/types/flying.png)  
**Category:** ![physical](../img/types/physical.png)  
**Power:** 90  
**Accuracy:** 95  
**PP:** 15  

## Description
User flies high into the air, dodging all attacks, and hits next turn.
