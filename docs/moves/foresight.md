# Foresight

**Type:** <img src="img/types/normal.png" width="60" />  
**Category:** <img src="img/types/status.png" width="50" />  
**Power:** -  
**Accuracy:** -  
**PP:** 40  

## Description
Forces the target to have no Evade, and allows it to be hit by Normal and Fighting moves even if it’s a Ghost.
