# Fury cutter

**TM/HM:** 

**Type:** ![bug](../img/types/bug.png)  
**Category:** ![physical](../img/types/physical.png){ style='object-fit:contain;' }  
**Power:** 40  
**Accuracy:** 95  
**PP:** 20  

## Description
Power doubles every turn this move is used in succession after the first, maxing out after five turns.
