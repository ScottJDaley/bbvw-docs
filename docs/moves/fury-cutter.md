# Fury cutter

**TM/HM:** 

**Type:** ![bug](../img/types/bug.png){ width=60 }  
**Category:** ![physical](../img/types/physical.png){ width=50 style='object-fit:contain;' }  
**Power:** 40  
**Accuracy:** 95  
**PP:** 20  

## Description
Power doubles every turn this move is used in succession after the first, maxing out after five turns.
