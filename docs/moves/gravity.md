# Gravity

**Type:** ![psychic](../img/types/psychic.png)  
**Category:** ![status](../img/types/status.png)  
**Power:** None  
**Accuracy:** None  
**PP:** 5  

## Description
Disables moves and immunities that involve flying or levitating for five turns.
