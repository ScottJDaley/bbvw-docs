# Healing wish

**Type:** ![psychic](../img/types/psychic.png)  
**Category:** ![status](../img/types/status.png)  
**Power:** None  
**Accuracy:** None  
**PP:** 10  

## Description
User faints.  Its replacement has its HP fully restored and any major status effect removed.
