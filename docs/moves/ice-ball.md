# Ice ball

**TM/HM:** 

**Type:** ![ice](../img/types/ice.png){ width=60 }  
**Category:** ![physical](../img/types/physical.png){ width=50 style='object-fit:contain;' }  
**Power:** 30  
**Accuracy:** 90  
**PP:** 20  

## Description
Power doubles every turn this move is used in succession after the first, resetting after five turns.
