# Memento

**Type:** ![dark](../img/types/dark.png)  
**Category:** ![status](../img/types/status.png)  
**Power:** None  
**Accuracy:** 100  
**PP:** 10

Lowers the target’s Attack and Special Attack by two stages.  User faints.
