# Miracle eye

**Type:** ![psychic](../img/types/psychic.png)  
**Category:** ![status](../img/types/status.png)  
**Power:** None  
**Accuracy:** None  
**PP:** 40  

## Description
Forces the target to have no evasion, and allows it to be hit by Psychic moves even if it’s Dark.
