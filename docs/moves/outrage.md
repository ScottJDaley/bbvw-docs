# Outrage

**TM/HM:** 

**Type:** ![dragon](../img/types/dragon.png){ width=60 }  
**Category:** ![physical](../img/types/physical.png){ width=50 style='object-fit:contain;' }  
**Power:** 120  
**Accuracy:** 100  
**PP:** 10  

## Description
Hits every turn for 2-3 turns, then confuses the user.
