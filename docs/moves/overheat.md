# Overheat

**Type:** ![fire](../img/types/fire.png)  
**Category:** ![special](../img/types/special.png)  
**Power:** 130  
**Accuracy:** 90  
**PP:** 5  

## Description
Lowers the user’s Special Attack by two stages after inflicting damage.
