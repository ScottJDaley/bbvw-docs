# Overheat

**TM/HM:** TM50

**Type:** <img src="../img/types/fire.png" width="60" />  
**Category:** <img src="../img/types/special.png" width="50" style="object-fit:contain;" />  
**Power:** 130  
**Accuracy:** 90  
**PP:** 5  

## Description
Lowers the user’s Special Attack by two stages after inflicting damage.
