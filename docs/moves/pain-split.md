# Pain split

**Type:** ![normal](../img/types/normal.png)  
**Category:** ![status](../img/types/status.png)  
**Power:** None  
**Accuracy:** None  
**PP:** 20  

## Description
Sets the user’s and targets’s HP to the average of their current HP.
