# Rapid spin

**TM/HM:** 

**Type:** ![normal](../img/types/normal.png)  
**Category:** ![physical](../img/types/physical.png){ style='object-fit:contain;' }  
**Power:** 50  
**Accuracy:** 100  
**PP:** 40  

## Description
Frees the user from binding moves, removes Leech Seed, and blows away Spikes.
