# Rapid spin

**Type:** <img src="img/types/normal.png" width="60" />  
**Category:** <img src="img/types/physical.png" width="50" style="object-fit:contain;" />  
**Power:** 50  
**Accuracy:** 100  
**PP:** 40  

## Description
Frees the user from binding moves, removes Leech Seed, and blows away Spikes.
