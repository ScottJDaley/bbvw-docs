# Rollout

**TM/HM:** 

**Type:** ![rock](../img/types/rock.png)  
**Category:** ![physical](../img/types/physical.png){ style='object-fit:contain;' }  
**Power:** 30  
**Accuracy:** 90  
**PP:** 20  

## Description
Power doubles every turn this move is used in succession after the first, resetting after five turns.
