# Skull bash

**Type:** ![normal](../img/types/normal.png)  
**Category:** ![physical](../img/types/physical.png)  
**Power:** 130  
**Accuracy:** 100  
**PP:** 10  

## Description
Raises the user’s Defense by one stage.  User charges for one turn before attacking.
