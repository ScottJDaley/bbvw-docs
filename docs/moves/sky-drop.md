# Sky drop

**TM/HM:** TM58

**Type:** ![flying](../img/types/flying.png)  
**Category:** ![physical](../img/types/physical.png){ style='object-fit:contain;' }  
**Power:** 60  
**Accuracy:** 100  
**PP:** 10  

## Description
Carries the target high into the air, dodging all attacks against either, and drops it next turn.
