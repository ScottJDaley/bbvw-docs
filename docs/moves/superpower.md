# Superpower

**TM/HM:** 

**Type:** ![fighting](../img/types/fighting.png){ width=60 }  
**Category:** ![physical](../img/types/physical.png){ width=50 style='object-fit:contain;' }  
**Power:** 120  
**Accuracy:** 100  
**PP:** 5  

## Description
Lowers the user’s Attack and Defense by one stage after inflicting damage.
