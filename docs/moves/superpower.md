# Superpower

**TM/HM:** 

**Type:** ![fighting](../img/types/fighting.png)  
**Category:** ![physical](../img/types/physical.png){ style='object-fit:contain;' }  
**Power:** 120  
**Accuracy:** 100  
**PP:** 5  

## Description
Lowers the user’s Attack and Defense by one stage after inflicting damage.
