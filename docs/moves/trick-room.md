# Trick room

**Type:** ![psychic](../img/types/psychic.png)  
**Category:** ![status](../img/types/status.png)  
**Power:** None  
**Accuracy:** None  
**PP:** 5  

## Description
For five turns, slower Pokémon will act before faster Pokémon.
