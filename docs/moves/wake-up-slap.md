# Wake up slap

**Type:** ![fighting](../img/types/fighting.png)  
**Category:** ![physical](../img/types/physical.png)  
**Power:** 70  
**Accuracy:** 100  
**PP:** 10  

## Description
If the target is asleep, has double power and wakes it up.
