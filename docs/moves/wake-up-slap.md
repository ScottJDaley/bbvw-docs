# Wake up slap

**TM/HM:** 

**Type:** ![fighting](../img/types/fighting.png){ width=60 }  
**Category:** ![physical](../img/types/physical.png){ width=50 style='object-fit:contain;' }  
**Power:** 70  
**Accuracy:** 100  
**PP:** 10  

## Description
If the target is asleep, has double power and wakes it up.
