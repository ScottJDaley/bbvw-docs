# Whirlpool

**TM/HM:** 

**Type:** ![water](../img/types/water.png)  
**Category:** ![special](../img/types/special.png){ style='object-fit:contain;' }  
**Power:** 35  
**Accuracy:** 85  
**PP:** 15  

## Description
Prevents the target from leaving battle and inflicts 1/16 its max HP in damage for 2-5 turns.
